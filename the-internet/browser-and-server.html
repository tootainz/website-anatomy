Browser and Server

aka. Frontend and backend

Usually a web page is mainly considered as just everything that gets sent to you, but itâ€™s actually a combination of what you receive to the browser (frontend) and what the server does when you send requests to is (backend)

Browser

A browser is a computer application that runs on your computer.

A browser is needed to navigate the internet into different addresses

A browser makes requests to servers

Servers then send the website code to the browser and the browser shows the actual visual website

Server

A server is a computer that runs a specific computer program that is supposed to receive requests and then to do different things based on the requests. Often these things are to send a specific website page but they can also be used to update information on an already sent website or to save information into a store called a database

Browsers and servers are like inseparable partners. The browser asks the server for a book page that you want to read and the server folds it, sends it like a paper plane over the internet and then the browser unfolds it and shows it to the reader.
