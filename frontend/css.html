CSS

CSS is a language that specifies the style of the html document. Basically, how the elements of the document should look like and how they should be positioned, like paragraph styles in a text editor

Basically, you have all this text and these images that you received inside your paper plane, now you cut all them out and CSS is the instructions, how you should organize them on your desk and what color you should use for them etc.

This information could be directly written to the document but it is cleaner to write it somewhere else

Examples of CSS

  • font-face
  • font size
  • color
  • background color
  • position
  • display
  • margin
  • padding
  • border
  • grid
